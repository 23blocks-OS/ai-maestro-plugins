# AI Maestro Plugin

AI agent orchestration plugin for Claude Code providing memory search, code graph queries, AMP messaging, docs search, and task planning.

## Skills

- **memory-search** - Search conversation history (requires AI Maestro service)
- **docs-search** - Search auto-generated documentation (requires AI Maestro service)
- **graph-query** - Query code graph relationships (requires AI Maestro service)
- **agent-messaging** - Send/receive messages via AMP protocol (requires AI Maestro service)
- **ai-maestro-agents-management** - Manage AI agents (requires AI Maestro service)
- **planning** - Complex task planning with persistent markdown files (standalone)

## Hooks

Session tracking and message notifications via `hooks.json`.

## Scripts

39 CLI scripts for memory, graph, docs, messaging, and agent management.
